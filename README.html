Quick Start

  1. Lancer le script configure.sh et suivre les instructions
  2. Importer le fichier gentable.sql dans la base de données utilisée pour avoir uniquement les tables OU le fichier jeu_de_test.sql pour avoir les tables remplies.
  3. Rendez vous à la racine du projet depuis votre navigateur et découvrez notre merveilleuse application.

Guide d'utilisation

Les onglets

  • Manifestation : Cet onglet vous permet d'accéder à toutes les manifestations présentes, d'en ajouter de nouvelles, de les modifier ou bien d'en supprimer.
  • IUT : Cet onglet vous permet d'accéder à tous les iut recensés dans notre base de données, d'en ajouter, de les modifier ou d'en supprimer.
  • Étudiants : Cet onglet vous permet de consulter les étudiants présent dans notre base de données, d'en ajouter, de les modifier ou d'en supprimer.
  • Épreuves : Cet onglet vous permet de consulter les épreuves disponibles, de les modifier, d'en ajouter ou bien d'en supprimer.

Les actions spéciales

  • → Cliquer sur cette icone vous permettra d'accéder à la liste des épreuves présentes dans une manifestation.

  • → Cliquer sur cette icone vous permettra d'accéder à la liste des étudiants par épreuves.

  • → Cliquer sur cette icone vous permettra d'accèder au profil d'un étudiant.

Les boutons sur la dernière colonne permettent respectivement de modifier, supprimer ou accéder à la fonction spéciale suivant l'ongler.

Le bouton modifier va faire apparaître le formulaire prérempli de la ligne séléctionnée.

Certaines têtes de colonnes permettent de classer par ordre alphabétique ou désalphabétique par le contenu de la colonne cliquée. Les colonnes cliquables afficheront un curseur de lien si elles sont activables.
